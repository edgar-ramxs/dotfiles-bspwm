# Instalación
```bash
sudo apt update -y && sudo apt upgrade -y
sudo apt install -y git vim
git clone https://github.com/edgar-ramxs/dotfiles-bspwm.git
cd dotfiles-bspwm
chmod +x installer.sh
./installer.sh
```
